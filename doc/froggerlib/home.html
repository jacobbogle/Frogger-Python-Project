Home¶

A Home object represents the spaces in the top row of the window separating the Grass spaces. The player wins when the frog reaches a Home space.

class Home(x, y, w, h)¶
x¶

The position of the home space in the x dimension; its left edge.

y¶

The position of the home space in the y dimension; its top edge.

w¶

The width of the home space.

h¶

The height of the home space.

type h

int or float

hits(other)¶

Determines if the given object (other) hits or touches the home space (self), according to both objects’ position, width, and height.

Hint: the other object should be the frog.

Parameters

other (Locatable) – The other object to be compared with.

Returns

Whether or not the other object touches the current object.

Return type

bool
