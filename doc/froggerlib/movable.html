Movable¶

Defines how objects move in the game. All classes that rely on movement inherit (whether directly or indirectly) from the Movable class which provides basic movement primitives.

You shouldn’t directly create an instance of Movable but consider the descriptions below of the data attributes/members and methods that are accessible on any object that inherits from Movable.

class Movable¶
desired_x¶

A value in the x dimension toward which the object will move.

desired_y¶

A value in the y dimension toward which the object will move.

speed¶

How quickly the object will move.

ride¶

Another object on which the current object is riding.

outOfBounds(screen_width, screen_height)¶

Reports whether or not the object has moved off of the screen. This is determined based on the screen size parameters.

Parameters

  • screen_height (int or float) – The height of the screen/window.
