Locatable¶

All objects should be located in the game window. Therefore, each class inherits (whether directly or indirectly) from the Locatable class which provides basic locating primitives.

You shouldn’t directly create an instance of Locatable but consider the descriptions below of the data attributes/members and methods that are accessible on any object that inherits from Locatable.

containsPoint(x, y)¶

Determines if the given point is contained within the space of the current object (self), according to the object’s position, width, and height.

Parameters

  • x (int or float) – The x value of the point to test.

  • y (int or float) – The y value of the point to test.

Returns

Whether or not the point is in the object.

Return type

bool

containsLocatable(other)¶

Determines if the given object is contained within the space of the current object (self), according to both objects’ position, width, and height.

Parameters

Returns
