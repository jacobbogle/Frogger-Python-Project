FroggerLib Objects¶

We provide a selection of class definitions for you to use in implementing your game. They are structured into a single package called froggerlib that you can import like this:

import froggerlib

Use this package as a namespace to any of the classes that you use when building your game. For example, you may access the Frog class like this (arguments omitted):

froggerlib.Frog(...)
